1. 出力形式は以下の通りです。parse失敗の原因となるので必ずJSONで出力してください。
```
{
    "議論": "(discussion)",
    "結論": {"agentIdx":(agent_idx)}
}
```
(discussion)には、勝利を目指す上で最良な襲撃先を決めるための**簡潔な**議論（つまり、口には出さない頭の中の思考）が、(agent_idx)には襲撃するAgentのagent_idx（int）が入ります。
2. 勝利を目指すこと。（勝利条件は割り当てられた役職によって異なることに注意すること）
3. 後述する人狼ゲームのルール、役職の詳細を必ず確認すること。
